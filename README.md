# Chick Quiz — A Cute Little Personality Quiz

## Project Description
Chick Quiz is a cozy mini quiz app where you answer simple questions and get a fun result at the end.

The visual style is light, playful, and intentionally simple.

This project was created as part of the **JavaScript course by Canshecode**, with a focus on practicing core JavaScript concepts.

**Technologies used:**
- HTML5
- CSS3 (custom styling, Flexbox layout, responsive design, media queries)
- JavaScript (DOM manipulation, events, basic logic)

**Planned future improvements:**
- Smooth transitions between questions
- Restart button and improved result screen
- Additional quiz themes
- LocalStorage support to remember the last result

## How to Use
The app is intentionally simple and intuitive:

**Choose an answer:**
- tap or click on an option

**Move to the next question:**
- click the button to continue

**See your result:**
- after the final question, the app displays your quiz result

## Mobile Support
Chick Quiz is adapted for mobile devices with a responsive layout and tap-friendly answer options.
